# Model Artifacts

This folder contains model artifacts generated by the training pipeline.

## Files

- `best_model.pkl` — Serialized best estimator pipeline selected from logistic regression, random forest, XGBoost, or LightGBM.
- `training_report.json` — Training metrics for the selected model.

## Training

Run the training script to generate artifacts:

```bash
cd model
python train.py
```

The script uses the sample dataset located at `dataset/stroke_data_sample.csv`. For production deployment, replace this file with a full clinical dataset and retrain the model.
